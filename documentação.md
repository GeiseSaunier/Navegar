# Navegar 

*Problema:* a venda de passagens é realizada de forma manual em todo o Amazonas. Isso demanda muito tempo.

*Justificativa:* tentar de alguma forma automatizar o processo.

*Solução viável:* desenvolvimento de um sistema, intitulado navegar, para solucionar essse problema.

    O navegar possui um adminstrador, reponsável pelos seguintes cadastros: trechos, dados do cliente e cidades. Além disso, efetuará a venda das passagens.
    Aliado a isso, temos o cliente, onde este poderá fazer a busca por trechos e a compra das passagens.

    Após o administrador cadastrar os trechos, o usuário poderá fazer a busca por destino e origem deste, escolhendo o trecho mais viável.

    Após a escolha do trecho, o usuário poderá solicitar a compra de passagens.
